What RTMify Exposes

RTMify now exposes MCP tools, resources, and prompts. Tools are best for filtered queries. Resources are stable read models. Prompts are curated workflows.

Example Resources
requirement://REQ-001 gap://1203/REQ-001 design-history://REQ-001 impact://UN-001 report://status
Example Prompts
trace_requirement(id="REQ-001") impact_of_change(id="UN-001") explain_gap(code=1203,node_id="REQ-001") audit_readiness_summary(profile="aerospace")
